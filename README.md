# Shadow-Dash-Endless-Runner-Game
A cyberpunk-inspired 2D endless runner game built with Python 3.9 and Pygame.

The game features procedural obstacle generation, rule-based AI difficulty scaling, a toggleable A* AI Runner demo mode, shield/magnet/slow-motion power-ups, physics simulation, particle effects, fullscreen presentation, persistent high scores, and modular game architecture.

The A* runner uses short-horizon forward planning with cached plans, safety-inflated collision checks, obstacle-aware action costs, and an emergency fallback for close threats.

Developed as an AI Lab university project demonstrating game AI concepts and clean software engineering principles.
